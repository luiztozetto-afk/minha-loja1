# Minha Loja Virtual

Este é um projeto de uma loja virtual feita com HTML, CSS e JavaScript.

## Funcionalidades
- Lista de produtos
- Botão de comprar
- Carrinho simples com contador

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Como usar
Basta abrir o arquivo index.html no navegador ou usar o Live Server no VS Code.
